# 更新日志

## 未发布

暂无。

## 0.1.4

用户数据维护版本。

### 新增

- 存储维护：查看用户数据目录、导出、备份、日志占用。
- 清理 30 天前旧导出和旧反馈包，不删除备份和聊天原文。
- 用户数据维护说明文档。

## 0.1.3

客户支持能力版本。

### 新增

- 客户反馈包：一键打包诊断信息、日志尾部、操作记录和环境信息，不包含聊天正文。
- 桌面端检查更新：比较本地版本与 GitHub Releases 最新版本，并可打开下载页面。

## 0.1.2

安装包发布能力版本。

### 新增

- Inno Setup 安装包工程，支持开始菜单快捷方式、可选桌面快捷方式和卸载入口。
- 本地安装包打包脚本 `打包安装包.ps1`。
- 简体中文安装器语言文件和第三方翻译授权文件。
- 安装包静默安装/卸载冒烟测试。
- GitHub Release 工作流自动上传绿色版 EXE 和安装包。

## 0.1.1

客户级部署稳定性版本。

### 新增

- 诊断中心：显示路径状态、会话数量、最近备份、最近操作。
- 复制诊断信息。
- 一键打开导出目录、备份目录、日志目录。
- 应用异常日志 `logs/应用日志.log`。
- 诊断中心与用户引导开发规划文档。
- 客户级用户数据目录，默认写入 `%LOCALAPPDATA%\CodexHistoryManager`。
- Codex 数据目录配置持久化，选择后下次启动自动沿用。
- 首次启动和无数据目录引导。
- 客户级产品化交付规划文档。

### 改进

- 后台任务异常会写入应用日志。
- 错误弹窗增加下一步排查建议。
- 导出、备份、日志不再依赖程序所在目录，提升 EXE 在客户电脑上的稳定性。

## 0.1.0

初始公开版本。

### 新增

- Windows 桌面版 Codex 本地对话历史管理器。
- 本地会话列表、筛选、预览、全文搜索。
- 会话归档、恢复、批量管理。
- 根据 JSONL 尾部消息修复 Codex 会话索引和排序。
- Markdown、HTML、TXT、JSON 导出。
- 图片附件提取。
- 写操作前自动备份。
- 备用本地 Web 管理界面。
- 公开仓库自检测试。
- GitHub Actions CI。
- GitHub Releases 自动构建 Windows EXE。
